## Usage

Use `.avatar`, `.avatar-sm`, or `.avatar-lg` on a small inline container. Put the image first and a fallback `<span>` after it.

```html
<span class="avatar">
  <img src="https://github.com/shadcn.png" alt="@shadcn" />
  <span>CN</span>
</span>
```

### HTML structure

<dl>
  <dt><code class="highlight language-html">&lt;div class="avatar-group"&gt;</code> <span class="badge-secondary">Optional</span></dt>
  <dd>
    Optional parent that groups avatars with overlap.
    <dl>
      <dt><code class="highlight language-html">&lt;span class="avatar|avatar-sm|avatar-lg"&gt;</code></dt>
      <dd>
        Avatar root. Use <code>.avatar-sm</code>, <code>.avatar</code>, or <code>.avatar-lg</code>. Use it standalone or as a direct child of <code>.avatar-group</code>.
        <dl>
          <dt><code class="highlight language-html">&lt;img&gt;</code> <span class="badge-secondary">Optional</span></dt>
          <dd>The avatar image. Provide meaningful <code>alt</code> text.</dd>
          <dt><code class="highlight language-html">&lt;span&gt;</code></dt>
          <dd>Fallback initials or icon.</dd>
          <dt><code class="highlight language-html">&lt;span data-badge&gt;</code> <span class="badge-secondary">Optional</span></dt>
          <dd>Status badge positioned at the block-end inline-end corner.</dd>
        </dl>
      </dd>
      <dt><code class="highlight language-html">&lt;span data-count&gt;</code> <span class="badge-secondary">Optional</span></dt>
      <dd>Count or icon indicator displayed at the end of the group.</dd>
    </dl>
  </dd>
</dl>
