## Usage

```html
<button type="button" class="btn">Primary</button>
<button type="button" class="btn-outline">Outline</button>
<button type="button" class="btn-sm-secondary">Small secondary</button>
<button type="button" class="btn-icon-outline" aria-label="Settings">
  <svg class="lucide lucide-settings" xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" ><path d="M9.671 4.136a2.34 2.34 0 0 1 4.659 0 2.34 2.34 0 0 0 3.319 1.915 2.34 2.34 0 0 1 2.33 4.033 2.34 2.34 0 0 0 0 3.831 2.34 2.34 0 0 1-2.33 4.033 2.34 2.34 0 0 0-3.319 1.915 2.34 2.34 0 0 1-4.659 0 2.34 2.34 0 0 0-3.32-1.915 2.34 2.34 0 0 1-2.33-4.033 2.34 2.34 0 0 0 0-3.831A2.34 2.34 0 0 1 6.35 6.051a2.34 2.34 0 0 0 3.319-1.915" /><circle cx="12" cy="12" r="3" /></svg>
</button>
```

Use exactly one `btn` class on a `<button>` or `<a>`. The class name combines size, icon shape, and variant:

| Pattern | Example | Use for |
| --- | --- | --- |
| `btn` | `btn` | Default-size primary button |
| `btn-{variant}` | `btn-outline` | Default-size button with a visual variant |
| `btn-{size}` | `btn-sm` | Sized primary button |
| `btn-{size}-{variant}` | `btn-sm-outline` | Sized button with a visual variant |
| `btn-icon` | `btn-icon` | Default-size icon-only primary button |
| `btn-icon-{variant}` | `btn-icon-outline` | Default-size icon-only button with a visual variant |
| `btn-{size}-icon` | `btn-sm-icon` | Sized icon-only primary button |
| `btn-{size}-icon-{variant}` | `btn-sm-icon-outline` | Sized icon-only button with a visual variant |

Do not stack button classes like `btn-sm btn-outline`. Use the composed class, such as `btn-sm-outline`.

### Variants

| Variant | Classes |
| --- | --- |
| Primary | `btn`, `btn-primary` |
| Secondary | `btn-secondary` |
| Outline | `btn-outline` |
| Ghost | `btn-ghost` |
| Link | `btn-link` |
| Destructive | `btn-destructive` |

Every variant can be combined with a size or icon shape. For example: `btn-xs-secondary`, `btn-lg-destructive`, `btn-icon-ghost`, `btn-sm-icon-outline`.

### Sizes

| Size | Text button | Icon-only button |
| --- | --- | --- |
| Extra small | `btn-xs` | `btn-xs-icon` |
| Small | `btn-sm` | `btn-sm-icon` |
| Default | `btn` | `btn-icon` |
| Large | `btn-lg` | `btn-lg-icon` |

Use icon-only classes only when the button has no visible text. Icon-only buttons must have an accessible name, usually with `aria-label`.

Mark inline icons with `data-icon="inline-start"` or `data-icon="inline-end"` when a text button should use icon-aware spacing.

To group related buttons, use the [Button Group](/components/button-group/) component.
