## Usage

Add `class="input"` to checkbox inputs. Pair it with [Field](/components/field) and a native `<label>` for layout and labeling.

```html
<input type="checkbox" class="input" />
```

### RTL

Checkbox layout uses document direction and logical spacing. Set `dir="rtl"` on a parent to render right-to-left.

<Preview>
<div dir="rtl" class="field" data-orientation="horizontal">
  <input type="checkbox" id="checkbox-rtl" class="input" aria-describedby="checkbox-rtl-description" checked />
  <section>
    <label for="checkbox-rtl">قبول الشروط والأحكام</label>
    <p id="checkbox-rtl-description">يمكنك تغيير هذا الخيار في أي وقت.</p>
  </section>
</div>
</Preview>
